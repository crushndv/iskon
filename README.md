# ИСКОН — Управление сайтом

## Быстрый старт: как добавить новый дом

1. Откройте браузер → перейдите на `https://ваш-сайт.ru/admin`
2. Войдите: email + пароль (настроены при первом запуске)
3. В меню слева нажмите **«🏠 Каталог домов»** → **«Все дома»**
4. Нажмите **«Add homes»** (Добавить дом)
5. Заполните поля (объяснения есть прямо под каждым полем)
6. Загрузите фото кнопкой **«Upload»**
7. Нажмите **«Publish»** (Опубликовать)
8. Подождите ~60 секунд → дом появится на сайте и карте

## Первоначальная настройка (делается один раз)

### 1. GitHub

1. Создайте аккаунт на [github.com](https://github.com) (если нет)
2. Создайте новый репозиторий: **New repository** → Название `iskon-site`
3. Загрузите все файлы этой папки

### 2. Netlify (хостинг — бесплатно)

1. Зайдите на [netlify.com](https://netlify.com) → **Sign up** через GitHub
2. Нажмите **Add new site** → **Import an existing project** → выберите `iskon-site`
3. Build command: *(пусто)*, Publish directory: `.` → **Deploy**
4. Сайт появится по адресу вида `iskon-site.netlify.app`
5. В **Site settings → Domain** подключите ваш домен (iskon.pro)

### 3. Панель управления (Netlify CMS)

1. В Netlify Dashboard: **Identity** → **Enable Identity**
2. **Settings & usage** → **Registration** → **Invite only**
3. **Services** → **Git Gateway** → **Enable Git Gateway**
4. **Identity** → **Invite users** → введите ваш email
5. Проверьте почту → нажмите ссылку → задайте пароль
6. Теперь `/admin` работает с вашим логином

### 4. Яндекс Карта

1. Зайдите на [developer.tech.yandex.ru](https://developer.tech.yandex.ru)
2. Создайте приложение → получите API-ключ (бесплатно)
3. Откройте файл `catalog.html`
4. Найдите строку: `apikey=YOUR_YANDEX_MAPS_KEY`
5. Замените `YOUR_YANDEX_MAPS_KEY` на ваш ключ
6. Сохраните и загрузите обновлённый файл в GitHub

### 5. Telegram-бот (уведомления о заявках)

1. Найдите в Telegram `@BotFather` → `/newbot` → получите токен
2. Узнайте ваш Chat ID: напишите боту и откройте `https://api.telegram.org/bot<TOKEN>/getUpdates`
3. В Netlify Dashboard: **Site settings → Environment variables**:
   - `TELEGRAM_BOT_TOKEN` = токен от BotFather
   - `TELEGRAM_CHAT_ID`   = ваш Chat ID

### 6. Bitrix24 (заявки в CRM)

1. В вашем Bitrix24: **Приложения** → **Входящий вебхук**
2. Выберите права: `crm` → скопируйте URL вебхука
3. В Netlify: **Environment variables** → `BITRIX24_WEBHOOK_URL` = URL вебхука

## Как редактировать тексты страниц

Для редактирования заголовков, описаний и текстов на страницах:

1. Откройте `/admin` → раздел **«⚙️ Настройки сайта»** (для контактов и телефона)
2. Для текстов на страницах — откройте `.html` файл в GitHub,
   найдите нужный текст и измените его. GitHub позволяет редактировать файлы прямо в браузере.

## Частые вопросы

**Как изменить телефон на сайте?**
Откройте `/admin` → Настройки сайта → Контакты → измените номер телефона.
Или в GitHub откройте `data/contacts.json` и отредактируйте.

**Как обновить фото дома?**
В `/admin` → Каталог домов → выберите дом → удалите старое фото → загрузите новое.

**Как снять дом с продажи?**
В `/admin` → Каталог → дом → поле «Статус» → выберите «Продан».

**Как добавить новый тип технологии?**
Откройте `admin/config.yml` в GitHub, найдите список `options` под полем `technology`
и добавьте новый вариант.

**Сайт не обновился после изменений?**
В Netlify Dashboard нажмите **Deploys** → **Trigger deploy** → **Deploy site**.
