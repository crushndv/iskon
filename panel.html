🚀 GitHub — авто-публикация

Как работает: При нажатии «Опубликовать» данные автоматически обновляются в вашем GitHub-репозитории, и через ~60 секунд сайт обновляется.
Токен: github.com/settings/tokens → New token → выберите scope repo.

Хранится только в вашем браузере
📞 Контакты сайта
Экспорт данных
Скачайте JSON и загрузите на GitHub вручную
Рекомендуем: настройте GitHub в разделе «Настройки» и нажимайте «Опубликовать» — всё обновится автоматически.
Скачать файлы вручную
Инструкция
  1. Скачайте файл кнопкой выше
  2. Откройте github.com → ваш репозиторий → папка data/
  3. Нажмите на файл → карандаш ✏️
  4. Выделите всё (Cmd+A) → вставьте содержимое нового файла
  5. Нажмите Commit changes
  6. Через ~60 сек сайт обновится
